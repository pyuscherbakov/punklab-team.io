# Описание

Документация для техниеской команды Punklab

## Разработка

### Зависимости

- Python3.12

### Установка зависимостей Python

``` shell
pip install poerty
poery install
```

### Запуск проекта

``` shell
mkdocs serve
```

## Деплой

``` shell
mkdocs build
```

После выполнения команды появится папка **site** с файлом **index.html**
